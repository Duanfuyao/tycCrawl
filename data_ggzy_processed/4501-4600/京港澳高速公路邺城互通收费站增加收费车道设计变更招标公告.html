title: 京港澳高速公路邺城互通收费站增加收费车道设计变更招标公告 time: 发布时间：2019-02-21 08:31 content: 京港澳高速公路邺城互通收费站增加收费车道设计变更招标公告 1. 招标条件 本招标项目京港澳高速公路邺城互通收费站增加收费车道设计变更已由邯郸市发展和改革委员会以/批准建设，施工图设计已由/以 /批准，项目业主为邯郸市漳河开发建设有限公司，建设资金来自自筹资金，出资比例为100%，招标人为 邯郸市漳河开发建设有限公司。项目已具备招标条件，现对该项目的施工进行公开招标。 2. 项目概况与招标范围 2.1 建设地点：京港澳高速公路邺城互通收费站2.2 建设规模：（1）收费广场采用入口侧单侧加宽方式，收费车道设置由“三进三出”调整为“四进六出”，其中入口方向增加MTC车道1条，出口方向增加MTC车道3条，相应增加收费岛、收费亭、交安设施、机电设备等，收费广场新增占地约11.7亩。（2）拆除入口方向现有收费大棚外挑部分及外侧造型构筑物，新建收费大棚采用框架结构，与现有收费大棚对称设置，屋顶采用坡屋面型式，外观风格与现有型式保持一致。（3）收费站出口至连接线AK1+050-AK1+158.471段路基断面宽度由12米调整为19.5米，对连接线BJK3+450-BJK3+820进行渐变改造，优化平面交叉设计，拼宽路面结构型式与现有型式保持一致。2.3 计划工期：3个月，缺陷责任期24个月，计划开工日期：2019年3月。2.4 招标范围: 施工图纸及工程量清单范围内所有工程施工。2.5 标段划分：本次招标分为1个施工标段。。 3. 投标人资格要求 3.1 本次招标要求投标人须具备详见附件3 资格审查条件资质，近5年内（2014年2月1日至今，以竣工或交工时间为准）至少独立承担过1项金额为1500万元人民币及以上的工程（公共建筑）施工业绩，工程质量合格为准业绩，并在人员、设备、 资金等方面具有相应的施工能力。 3.2 本次招标不接受联合体投标。 3.3 每个投标人最多可对1个标段投标；被招标项目所在地省级交通运输主管部门评为/信用等级的投标人，最多可对/个标段投标。每个投标人允许中/个标。对投标人信用等级的认定条件为：/。 3.4 与招标人存在利害关系可能影响招标公正性的单位，不得参加投标。单位负责人为同一人或存在控股、管理关系的不同单位，不得参加同一标段投标，否则，相关投标均无效。 3.5 在“信用中国”网站（http://www.creditchina.gov.cn/)中被列入失信被执行人名单的投标人，不得参加投标。 4.招标文件的获取 4.1 凡有意参加投标者，请于2019-02-21至2019-02-27，每日上午09:00时至12:00时，下午14:00时至17:00时（北京时间，下同），在邯郸市行政服务中心4楼持单位介绍信和经办人身份证购买招标文件。参加多个标段投标的投标人必须分别购买相应标段的招标文件，并对每个标段单独递交投标文件。 4.2 招标文件售价1000元，图纸每套售价1000元，招标人根据 对本合同工程勘察所取得的水文、地质、气象和料场分布、取土场、弃土场位置等资 料编制的参考资料每套售价 0 元，售后不退。 5. 投标文件的递交及相关事宜 5.1 招标人将于下列时间和地点组织进行工程现场踏勘并召开投标预备会。踏勘现场时间:，集中地点:/； 投标预备会时间:，地点:/； 5.2 投标文件递交的截止时间为2019-03-13 14:00 ，投标人应于当日13:30至14:00将投标文件递交至 邯郸市行政服务中心四楼4开标室 。 5.3 逾期送达的、未送达指定地点的或不按照招标文件要求密封的投标文件，招标人将予以拒收。 6. 发布公告的媒介 本次招标公告同时在河北省招标投标公共服务平台、河北省公共资源交易平台上发布。 7.联系方式 招标人： 邯郸市漳河开发建设有限公司 招标代理机构： 河北坛正工程项目管理有限公司 地址： 邯郸市磁县讲武城镇北白道村东 地址： 邯郸市丛台区滏河北大街396号华浩天际A座1412 邮编： 056000 邮编： 056000 联系人： 郝志忠 联系人： 姜萌 电话： 13831017912 电话： 15533003675 传真： / 传真： / 电子邮件： / 电子邮件： hbsj1015@163.com 网址： / 网址： / 开户银行： / 开户银行： / 账号： / 账号： / 附件:招标公告附件.rar
