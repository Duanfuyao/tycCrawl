title: 衡水市西外环北外环供热一次网改扩建工程项目设计 招标公告 time: 发布时间：2019-02-26 08:34 content: 衡水市西外环北外环供热一次网改扩建工程项目设计 招标公告 1. 招标条件 本招标项目衡水市西外环北外环供热一次网改扩建工程项目已由衡水市行政审批局以衡行审投资管理[2019]23号批准建设，项目业主为衡水恒通热力有限责任公司，建设资金来自自筹资金和银行贷款，出资比例为100%，招标人为 衡水恒通热力有限责任公司。项目已具备招标条件，现对该项目的设计进行公开招标。 2. 项目概况与招标范围 项目为西外环北外环一次网主管路敷设，起自西外环与人民路交口处向北，途经和平路、石德铁路、班曹店排干渠，到达北外环，再由北外环沿路向东，途经前进街、育才大街至中华大街处接原有管道。敷设管径为DN1200供热管网，沟槽总长度为7160米，供热面积约1392. 52万平方米。。 3. 投标人资格要求 3.1 本次招标要求投标人须具备建设行政主管部门核发的有效期内的市政行业（热力工程）专业设计甲级资质或市政行业设计甲级资质或工程设计综合甲级资质，建设行政主管部门核发的有效期内的市政行业（热力工程）专业设计甲级资质或市政行业设计甲级资质或工程设计综合甲级业绩，并具有与本招标项目相应的设计能力。 3.2 本次招标不接受联合体投标。 4. 技术成果经济补偿 本次招标对未中标人投标文件中的技术成果不给予经济补偿。给予经济补偿的，招标人将按如下标准支付经济补偿费用：0 5.招标文件的获取 5.1 凡有意参加投标者，请于2019-02-26至2019-03-03（北京时间，下同），登录冀招标交易平台下载电子招标文件。 5.2 招标文件售价0元，售后不退。技术资料押金0元，在退还技术资料时退还（不计利息）。 6. 投标文件的递交 6.1 投标文件递交的截止时间为2019-03-18 09:00:00投标人应在截止时间前通过网上递交递交电子投标文件。 6.2 逾期送达的投标文件，电子招标投标交易平台将予以拒收。 7. 发布公告的媒介 本次招标公告同时在河北省招标投标公共服务平台、衡水市公共资源交易信息平台上发布。 8.联系方式 招标人： 衡水恒通热力有限责任公司 招标代理机构： 河北海新工程项目管理有限公司 地址： 衡水市西外环北外环 地址： 无 邮编： 无 邮编： 无 联系人： 丁晋勇 联系人： 丁晋勇 电话： 0311-85860497 电话： 0311-85860497 传真： 无 传真： 无 电子邮件： 无 电子邮件： 无 网址： http://www.jizhaobiao.com 网址： http://www.jizhaobiao.com 开户银行： 无 开户银行： 无 账号： 无 账号： 无
