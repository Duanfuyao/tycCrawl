title: 衡水市冀州区第四中学等四所学校2018年校舍维修改造项目（冀州区徐家庄乡第一小学伙房、餐厅及消防工程）招标公告 time: 发布时间：2019-02-21 08:31 content: 衡水市冀州区第四中学等四所学校2018年校舍维修改造项目（冀州区徐家庄乡第一小学伙房、餐厅及消防工程）招标公告 1. 招标条件 本招标项目衡水市冀州区第四中学等四所学校2018年校舍维修改造项目（冀州区徐家庄乡第一小学伙房、餐厅及消防工程）已由衡水市冀州区发展改革创新局以冀州发改审批[2018]81号批准建设，招标人为衡水市冀州区徐家庄乡第一小学，建设资金来自财政资金，项目出资比例为100%。项目已具备招标条件，现对该项目的施工进行公开招标。 2. 项目概况与招标范围 施工图纸及工程量清单范围内的全部内容。 3. 投标人资格要求 3.1 本次招标要求投标人须具备投标人须具备建筑工程施工总承包三级及以上资质及消防设施工程专业承包二级及以上资质资质， 投标人须具备建筑工程施工总承包三级及以上资质及消防设施工程专业承包二级及以上资质业绩，并在人员、设备、资金等方面具有相应的施工能力，其中，投标人拟派项目经理须具备其他专业初级级注册建造师执业资格，具备有效的安全生产考核合格证书，且未担任其他在施建设工程项目的项目经理。 3.2 本次招标 不接受联合体投标。 3.3 各投标人均可就本招标项目上述标段中的1个标段投标，但最多允许中标1个标段。 4.投标报名 凡有意参加投标者，请于2019-02-21 09:00至2019-03-14 09:00(法定公休日、法定节假日除外)，登录网上报名报名 5.招标文件的获取 5.1 凡通过上述报名者，请于2019-02-21至2019-02-27(法定公休日、法定节假日除外)，登录冀招标交易平台下载电子招标文件。 5.2 招标文件每套售价0 元，售后不退。图纸押金0元，在退还图纸时退还（不计利息）。 6. 投标文件的递交 6.1 投标文件递交的截止时间为2019-03-14 09:00:00，，投标人应在截止时间前通过网上递交递交电子投标文件。 6.2 逾期送达的投标文件，电子招标投标交易平台将予以拒收。 7. 发布公告的媒介 本次招标公告同时在河北省招标投标公共服务平台、衡水市公共资源交易信息平台 上发布。 7.联系方式 招标人： 衡水市冀州区徐家庄乡第一小学 招标代理机构： 衡水正邦工程招标代理有限公司 地址： 衡水市冀州区徐家庄乡第一小学院内 地址： 无 邮编： 无 邮编： 无 联系人： 高银姗 联系人： 高银姗 电话： 0318-7179968 电话： 0318-7179968 传真： 无 传真： 无 电子邮件： 无 电子邮件： 无 网址： 无 网址： 无 开户银行： 无 开户银行： 无 账号： 无 账号： 无
