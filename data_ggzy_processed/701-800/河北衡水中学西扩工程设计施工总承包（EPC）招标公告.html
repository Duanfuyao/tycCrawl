title: 河北衡水中学西扩工程设计施工总承包（EPC）招标公告 time: 发布时间：2019-02-23 00:11 content: 河北衡水中学西扩工程设计施工总承包（EPC）招标公告 1.招标条件 本招标项目河北衡水中学西扩工程设计施工总承包（EPC）已由衡水市行政审批局以衡水市行政审批局关于衡水中学西扩项目招标方案核准的意见,衡行审投资管理【2019】32号批准建设，项目业主为河北衡水中学，建设资金来自财政资金，出资比例为100%，招标人为河北衡水中学。项目已具备招标条件，现对该项目的施工进行公开招标。 2.项目概况与招标范围 2.1项目名称：河北衡水中学西扩工程设计施工总承包（EPC）2.2项目建设地点：河北衡水中学西侧2.3建设规模及内容：新建占地面积146.8亩。主要新建内容：教学楼（能容纳2200人）、宿舍楼（能容纳2200人）、艺体中心、餐厅（能容纳2200人）、学生发展中心、地下停车场、运动场以及相关水电配套设施等。2.4工期：2019年7月20日前完成（包括设计、图审、施工准备、施工与竣工验收）2.5招标范围：完成本项目的设计、施工总承包。包括但不限于项目立项批复内的原方案设计深化、初步设计、施工图设计、概算编制及施工现场配合服务、以及施工图范围内的全部施工内容。达到满足使用功能，并达到合同约定的验收标准。2.6质量标准: 设计质量标准：符合现行设计规范及技术标准要求； 施工质量标准：达到国家现行有关工程质量验收规范合格标准；。 3.投标人资格要求 3.1 本次招标要求投标人须具备3.1投标人须具有独立法人资格，并在人员、设备、资金等方面具有相应的设计及施工能力。 3.2 本次招标要求投标人须同时具备设计与施工资质，其中： 设计资质要求为：建筑行业（建筑工程）设计甲级资质或工程设计综合甲级资质。 施工资质要求为：具备建设行政主管部门颁发的建筑工程施工总承包一级及以上资质。 本项目接受联合体投标，一个投标联合体的组成单位最多不能超过2家；一个单位只能参与组建一个联合体。联合体投标的，联合体各方须签订联合体协议书，明确联合体牵头人和各方的权利义务。牵头人为具备建设行政主管部门颁发的建筑工程施工总承包一级及以上的施工单位。 3.3投标人拟派项目经理须具备建筑工程专业壹级注册建造师资格，具备有效的安全生产考核合格证书B本。 3.4设计负责人资格要求：投标人拟派项目设计负责人须具备壹级注册建筑师注册证书； 3.4法定代表人或单位负责人为同一人或者存在控股、管理关系的不同单位，不得同时参加本项目投标，否则均按投标资格无效认定。 3.5资格审查方式：资格后审。资质， /业绩，并在人员、设备、资金等方面具有相应的施工能力。 3.2 本次招标 接受联合体投标。联合体投标的，应满足下列要求：本项目接受联合体投标，一个投标联合体的组成单位最多不能超过2家；一个单位只能参与组建一个联合体。联合体投标的，联合体各方须签订联合体协议书，明确联合体牵头人和各方的权利义务。牵头人为具备建设行政主管部门颁发的建筑工程施工总承包一级及以上的施工单位。 3.3 各投标人均可就上述标段中的1个标段投标。 4.招标文件的获取 4.1 凡有意参加投标者，请于2019-02-23至2019-02-27（北京时间，下同），登录招采进宝河北专区下载电子招标文件。 4.2 招标文件每套售价0 元，售后不退。技术资料押金0元，在退还技术资料时退还（不计利息）。 5. 投标文件的递交 5.1 投标文件递交的截止时间为2019-03-15 14:30:00，投标人应在截止时间前通过网上获取递交电子投标文件。 5.2 逾期送达的投标文件，电子招标投标交易平台将予以拒收。 6. 发布公告的媒介 本次招标公告同时在河北省招标投标公共服务平台、衡水市公共资源交易信息平台、招采进宝河北专区 上发布。 7.联系方式 招标人： 河北衡水中学 招标代理机构： 衡水建银工程造价事务所有限公司 地址： 无 地址： 无 邮编： 无 邮编： 无 联系人： 张雪一、张宗科 联系人： 薛金榜 电话： 0318-2126904 电话： 0318-8883897 传真： 无 传真： 无 电子邮件： 无 电子邮件： hsjy0318@163.com 网址： 无 网址： 无 开户银行： 无 开户银行： 无 账号： 无 账号： 无
