title: 河北衡水中学西扩工程监理招标公告 time: 发布时间：2019-02-23 00:11 content: 河北衡水中学西扩工程监理招标公告 1.招标条件 本招标项目河北衡水中学西扩工程监理已由 衡水市行政审批局以衡水市行政审批局关于衡水中学西扩项目招标方案核准的意见,衡行审投资管理【2019】32号批准建设，项目业主为河北衡水中学，建设资金来自财政资金，出资比例为100%，招标人为河北衡水中学。项目已具备招标条件，现对该项目的监理进行公开招标。 2.项目概况与招标范围 项目的建设地点：河北衡水中学西侧建设规模及内容：新建占地面积146.8亩。主要新建内容：教学楼（能容纳2200人）、宿舍楼（能容纳2200人）、艺体中心、餐厅（能容纳2200人）、学生发展中心、地下停车场、运动场以及相关水电配套设施等。监理服务周期：自监理合同签订生效之日起至工程质保期结束止招标范围：本项目施工及保修阶段全过程监理质量标准：合格。 3.投标人资格要求 3.1 本次招标要求投标人须具备3.1投标人须具有独立法人资格且同时具有建设行政主管部门核准和颁发的房屋建筑工程监理甲级资质、机电安装工程监理乙级及以上资质。 3.2总监理工程师须为国家注册监理工程师（建筑工程专业），且未担任其他在建工程项目的总监理工程师。 3.2法定代表人或单位负责人为同一人或者存在控股、管理关系的不同单位，不得同时参加本项目投标，否则均按投标资格无效认定。 3.3本次招标不接受联合体投标。 3.4资格审查方式：资格后审。资质，/业绩，并在人员、试验检测仪器设备方面具有相应的监理能力。 3.2 本次招标不接受联合体投标。 4.招标文件的获取 4.1 凡有意参加投标者，请于2019-02-23至2019-02-27（北京时间，下同），登录招采进宝河北专区下载电子招标文件。 4.2 招标文件每套售价0 元，售后不退。技术资料押金0元，在退还技术资料时退还（不计利息）。 5. 投标文件的递交 5.1 投标文件递交的截止时间为2019-03-15 14:00:00，投标人应在截止时间前通过网上获取递交电子投标文件。 5.2 逾期送达的投标文件，电子招标投标交易平台将予以拒收。 6. 发布公告的媒介 本次招标公告同时在河北省招标投标公共服务平台、衡水市公共资源交易信息平台、招采进宝河北专区 上发布。 7.联系方式 招标人： 河北衡水中学 招标代理机构： 衡水建银工程造价事务所有限公司 地址： 无 地址： 无 邮编： 无 邮编： 无 联系人： 张雪一、张宗科 联系人： 薛金榜 电话： 0318-2126904 电话： 0318-8883897 传真： 无 传真： 无 电子邮件： 无 电子邮件： hsjy0318@163.com 网址： 无 网址： 无 开户银行： 无 开户银行： 无 账号： 无 账号： 无
